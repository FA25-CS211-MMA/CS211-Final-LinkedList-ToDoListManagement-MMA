# CS211 Final Project – Linked List To-Do List Manager
Author: Mehmet Mert Asma

------------------------------------------------------------
DESCRIPTION
------------------------------------------------------------
This project implements a console-based To-Do List Manager using a singly linked list.
It allows the user to add, insert, delete, update, search, and view tasks dynamically.
Each operation demonstrates linked list manipulation such as node creation, traversal, and pointer updates.

------------------------------------------------------------
FEATURES
------------------------------------------------------------
- Add task at the beginning, end, or a specific position
- Delete task by title or by position
- Update an existing task at a specific position
- Search for a task by title
- Display all tasks in order
- Show total number of tasks
- Simple, menu-based console interface

------------------------------------------------------------
DATA STRUCTURE CHOICE
------------------------------------------------------------
A linked list was chosen as the core data structure because it allows efficient insertions and deletions
at any position without shifting elements like an array would. Each task is stored in a node containing
the task title and a pointer to the next node. This structure is ideal for a dynamic to-do list where users
frequently add, remove, or update tasks.
